# Frontend PRD
## ML-Based Resume Evaluator

---

## Purpose
Provide a simple single-user interface to upload job descriptions and resumes, trigger evaluation, and view ranked results.

## Functional Requirements
- Upload 1 Job Description file (PDF/DOCX/TXT)
- Upload multiple resume files
- Evaluate button
- Display loading indicator
- Show ranked results table

## Results Table Columns
| Rank | Candidate Name | Match Score |

## Non-Functional Requirements
- Responsive UI
- Fast loading
- Minimal design
- No authentication

## Tech Stack
- HTML
- CSS
- JavaScript (optional)
